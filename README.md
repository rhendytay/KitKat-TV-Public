# 📺 KitKat TV - Official Distribution

Selamat datang di repositori distribusi resmi **KitKat TV**. Repositori ini digunakan untuk menghosting file APK terbaru, catatan rilis (changelog), dan mendukung sistem pembaruan otomatis (*Auto-Update*) bagi pengguna KitKat TV.

## 🚀 Fitur Utama
- **Android TV Optimized**: Antarmuka yang bersih dan mudah digunakan dengan remote control.
- **Auto-Update**: Aplikasi akan memberitahu Anda secara otomatis jika ada versi terbaru.
- **Custom Channel Number**: Mendukung input nomor saluran langsung dari remote (M3U `tvg-chno`).
- **High Performance**: Menggunakan ExoPlayer versi terbaru untuk pemutaran stream yang stabil.
- **Lightweight**: Tanpa iklan dan sangat ringan.

## 🛠️ Cara Instalasi (Android TV / Box)
Karena aplikasi ini tidak tersedia di Google Play Store, Anda perlu melakukan instalasi manual (*Sideloading*):

1. **Unduh APK**: Ambil file `.apk` terbaru dari link di atas.
2. **Transfer file**: Pindahkan file APK ke TV menggunakan Flashdisk atau aplikasi seperti *Send Files to TV*.
3. **Izinkan Instalasi**:
   - Buka **Settings** di TV Anda.
   - Pilih **Security & Restrictions**.
   - Aktifkan **Unknown Sources** untuk File Manager yang Anda gunakan.
4. **Instal**: Buka file APK dan pilih **Install**.

---

## 🔄 Sistem Pembaruan Otomatis
Aplikasi ini dilengkapi dengan fitur cek versi otomatis. 
- Setiap kali aplikasi dibuka, ia akan mengecek file `release.json` di repositori ini.
- Jika versi baru tersedia, dialog pembaruan akan muncul.
- Anda cukup menekan **"Perbarui Sekarang"** untuk mengunduh dan menginstal versi terbaru secara otomatis.
